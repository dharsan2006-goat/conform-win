# Conform-Win Android App

Simple Android application for MeshWave LoRa communication.

## Features
- BLE connectivity to ESP32 LoRa Bridge
- Message sending and receiving
- Real-time communication

## Setup
1. Install Android Studio
2. Open this project
3. Build and install APK

## Usage
1. Connect to ESP32 via BLE
2. Send/receive messages through LoRa network
